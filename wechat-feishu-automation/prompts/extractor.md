# 微信公众号文章内容提取提示词

## 提取字段

| 字段名 | 说明 |
|--------|------|
| 标题 | 文章标题，原文提取 |
| 作者 | 公众号名称 |
| 发布时间 | 格式：YYYY-MM-DD |
| 正文 | 清洗后的正文 |
| 封面图 | 图片 URL |
| 原文链接 | 原始链接 |
| 采集时间 | 爬取时间戳 |

## 正文清洗规则

### 需要移除的内容

1. **广告内容**
   - 商品推荐
   - 品牌植入
   - 推广链接

2. **关注引导**
   - "长按识别二维码关注"
   - "点击上方蓝字关注"
   - "设为星标" 等

3. **冗余内容**
   - 多余空行
   - 连续空格
   - 无关符号

### 保留的内容

- 原创正文
- 段落结构
- 有意义的引用
- 相关话题标签

## 输出格式

```json
{
  "title": "文章标题",
  "author": "公众号名称",
  "publish_time": "2024-01-01",
  "content": "清洗后的正文内容...",
  "cover_image": "https://...",
  "original_url": "https://mp.weixin.qq.com/s/...",
  "scrape_time": "2024-01-01 12:00:00"
}
```
