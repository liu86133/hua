# 📕 小红书飞书运营自动化 Skill

小红书 + 飞书多维表格 自动化运营系统

## 功能特性

- ✅ **自动爬取**：从飞书表格「原文链接」自动爬取小红书笔记
- ✅ **AI创作**：基于原文和评论需求，AI生成爆款标题和正文
- ✅ **图片优化**：AI优化原图，生成高点击率封面
- ✅ **一键发布**：自动登录小红书，一键发布优化内容
- ✅ **数据同步**：发布结果自动同步回飞书表格

## 飞书表格字段

| 字段名 | 描述 |
|--------|------|
| 原文链接 | 填入小红书笔记URL触发自动化 |
| 原文标题 | 自动爬取 |
| 原文图片 | 自动爬取 |
| 原文正文 | 自动爬取 |
| 原文发布时间 | 自动爬取 |
| 原文点赞数 | 自动爬取 |
| 评论需求 | 从评论区提取用户需求 |
| 阅读数量 | 自动爬取 |
| 仿写标题 | AI生成5-10个热门标题 |
| 修改图片 | AI优化后的图片 |
| 仿写正文 | AI生成原创正文 |
| 一键发布 | 复选框触发发布 |

## 工作流程

```
飞书表格填写链接
       ↓
   内容爬取
       ↓
   字段提取
       ↓
   AI标题生成
       ↓
   AI正文创作
       ↓
   图片优化
       ↓
   用户确认勾选发布
       ↓
   自动发布到小红书
       ↓
   结果同步回飞书
```

## 安装

1. 将 `xiaohongshu-feishu-automation` 文件夹复制到 OpenClaw skills 目录
2. 安装依赖：
   ```bash
   pip install playwright
   playwright install chromium
   ```
3. 配置环境变量：
   ```bash
   export DOUBAO_API_KEY="your-doubao-api-key"
   ```
4. 重启 OpenClaw

## 使用方法

### 方式1：飞书多维表格触发

1. 在飞书多维表格中创建「小红书运营自动化」系统
2. 在「原文链接」列填入小红书笔记URL
3. OpenClaw 自动检测并执行全流程

### 方式2：飞书聊天指令

```
@机器人 爬取 https://www.xiaohongshu.com/explore/xxx
@机器人 AI改写
@机器人 一键发布
```

## 提示词模板

### 爆款标题生成

```
你是小红书爆款标题专家，基于以下信息生成5-10个高点击率标题：
- 原文标题：{title}
- 原文正文：{content}
- 用户评论需求：{comments}

要求：
1. 每个标题15-30字
2. 融入痛点/数字/情绪/悬念元素
3. 保留核心主题
4. 加入热门关键词
```

### 爆款正文生成

```
你是小红书爆款文案专家，基于以下信息生成1-3篇原创正文：
- 原文正文：{content}
- 用户评论需求：{comments}

要求：
1. 100%原创
2. 开头抓眼球 + 中间干货 + 结尾引导互动
3. 解决用户痛点
4. 加入话题标签#
5. 语气亲切自然
```

## 配置要求

- Python 3.8+
- Playwright (浏览器自动化)
- 豆包 API Key (AI生成)
- 飞书多维表格权限
- 小红书账号 Cookies
